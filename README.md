# EXTERNAL-SOURCE-TRANSFORM

EXTERNAL-SOURCE-TRANSFORM is a microservice that transforms json data from an external source to a desired format.

## Requirements

- Python 3.12 or higher
- PipEnv 2023.12.1 or higher

## Usage

```python
pipenv shell
pipenv install
python3 app.py
```
